# TMS FNC Dashboard Pack
O TMS FNC Dashboard Pack é um conjunto de componentes visuais para a criação de dashboards modernos e interfaces de monitoramento em tempo real no Delphi.

O pacote oferece indicadores de progresso, gauges, displays de LED e LCD, indicadores de tendência, gráficos de distribuição e outros widgets totalmente personalizáveis. Por fazer parte da família FNC, os componentes podem ser utilizados em aplicações VCL, FireMonkey e TMS WEB Core, permitindo compartilhar uma única base de código entre aplicações Windows, multiplataforma e Web.

Este repositório reúne exemplos práticos de utilização dos componentes do TMS FNC Dashboard Pack.
